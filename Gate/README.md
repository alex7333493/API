## 🚗 Кейс: Интеграция с системами распознавания номеров (ANPR / LPR)

**Для кого:** Команды, разрабатывающие софт для парковок, ЖКХ, бизнес-центров и умных дворов, где логика авторизации и распознавания номеров лежит на сервере или камере.

### Боль разработчиков софта:
Вы написали отличную нейросеть, которая за доли секунды распознает автомобильный номер с камеры. Но теперь вам нужно как-то физически открыть шлагбаум. Тянуть провода от сервера, ставить самодельные реле на базе Arduino через USB, которые постоянно зависают — это ненадежно и несерьезно для B2B-рынка.

### Решение от Unit-1:
Модуль **Unit-1** устанавливается прямо в тумбу шлагбаума или в ближайший монтажный шкаф, подключается к вашей локальной сети по **Ethernet** и управляет стрелой через **сухие контакты** (имитируя нажатие кнопки на пульте или кнопке охраны).

Ваш софт отправляет простой **POST-запрос** (или TCP-пакет) — Unit-1 мгновенно открывает проезд.

### 🛠️ Документация API для управления шлагбаумом

#### 1. Команда на открытие шлагбаума (Импульсное замыкание)
Обычно для открытия шлагбаума требуется не постоянное включение реле, а короткий импульс (замыкание «сухого контакта» на 0.5–1 секунду). Unit-1 умеет делать это аппаратно, гарантируя, что контакт не останется замкнутым.

* **Протокол:** HTTP (или чистый TCP)
* **Метод:** `POST`
* **URL:** `http://<unit1_ip>/api/v1/barrier/open`

**Тело запроса (JSON):**
```json
{
  "device_id": "UNIT1-PARK-01",
  "secret_key": "your_secure_api_token",
  "action": "pulse_open",
  "duration_ms": 500
}
